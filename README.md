# Ghost in the Machine: IPL Auction Analytics

## Overview
Quantifies bowler's 'killer instinct' using Bayesian inference on T20 data.

## Setup
1. Install Poetry if needed: `pip install poetry`
2. Run: `poetry install`
3. Activate: `poetry shell`
4. Launch: `jupyter notebook notebooks/analysis.ipynb`

## Data
Raw: data/raw/IPL_Bowler_Detailed_Data.xls
Processed: Generated in data/processed/

## Output
- Plots in docs/posteriors.png
- Summary in docs/executive_summary.txt (convert to PDF)
